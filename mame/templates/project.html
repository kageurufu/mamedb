The MAME Index Project

This project was started to give me an easy way to query the current mame game database for information on supported games, with different features sets. I am in the process of building a full size arcade machine to play with my friends, and needed to know how many buttons, what types of joysticks, and how many of what type I needed in order to comfortably play all my favorite games on one control panel. It then grew into a more generic system for full searching the mame database. Due to the huge amount of games that mame supports (whether working or not), There are thousands of ways to query this data, and find the games you are looking for.

This project is fully open source, down to MAME itself. Fork us on GitHub at http://github.com/KageUrufu/mamedb

Go ahead, run your own copy, make changes, have fun. This project is GPL v2, so just share your (awesome) work with the world!
